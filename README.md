# SparseEigensolver

Eigenvalue decomposition library.
Under development.

Aims.
- Eigenvalue decomposition for dense matrices.
- Davidson diagonalization for sparse matrices.
- MPI, CUDA and ROCm communicator suppoort.

Prerequisite:
- YAML-cpp
- Math kernel Library (cblas, LAPACKE, pblas)
- C++17 or higher
